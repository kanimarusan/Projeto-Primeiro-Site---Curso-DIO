HTML, é uma linguagem de programação?

Não. Não é.

HTML é um arquivo de marcação. Textos delimitados por nomes que os navegadores conseguem ler, interpretar e executar.

a sigla significa: HyperText Markup Language (Linguagem de Marcação de Hipertexto).

Ferramentas Utilizas & Tags

Para por em prática o estudo de HTML recomendo que utilize as seguintes ferramentas:

    Visual Studio Code: Para escrever os códigos HTML para os seus sites de estudo.

    O plugin Live Server: Plugin baixado dentro do Visual Studio para você visualizar seus projetos no seu navegador

Estrutura Básica

Com o Visual Studio aberto, crie uma pasta no seu computador com o nome do projeto e dentro do programa clique em Abrir Pasta

para localiza-la e iniciar o seu projeto. Em seguida, clique no ícone de arquivo com um sinal de "+" e escreva o comando

"index.html".

A partir daí montamos a estrutura básica. Primeiro com o comando HTML que define que se trata de uma página html.

Em seguida o comando HEAD que é o que pré carrega antes de exibir a página pro usuário. E por último o comando BODY

comando que define o que será exibido na página para o usuário quando carregada. Para aprofunda mais veja em Referência.
